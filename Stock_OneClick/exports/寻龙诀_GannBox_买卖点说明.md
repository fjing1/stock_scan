# 寻龙诀 Gann Box 与买卖点说明

更新时间：2026-05-19 16:04

## 1. 指标定位

这套指标不是单一买卖按钮，而是一个多周期辅助系统：

- 日线用于判断大环境和趋势背景。
- 4H 用于捕捉更及时的反弹点或做空点。
- Gann Box 的 0 / 1 用来定义一段行情的低点、高点和回撤位置。
- 寻龙诀之分金用于判断情绪位置：低位、修复、高位转弱。

核心思想：日线过滤方向，4H 提供触发。

## 2. Gann Box 的 0 点规则

0 点代表一段上涨结构的起点。

当前规则：

1. 指标在非上涨段中持续记录最低价，作为候选 0 点。
2. 当 EMA10 从下降或走平转为上升时，候选最低点被识别为临时 0 点。
3. 图表会在识别当天给出“0出”提示。
4. 0 点本身通常画在更早的最低价 K 线上，而“0出”标记显示的是系统真正识别出这个 0 的日期。

所以要区分：

- 0：结构低点的位置。
- 0出：系统确认这个低点开始具备反弹结构的时间。

## 3. Gann Box 的 1 点规则

1 点代表这段上涨结构的高点。

当前规则：

1. 0 点出现后，EMA10 持续上升，系统开始记录这一段的最高价。
2. 必须满足最低上涨条件：
   - EMA10 连续上升不少于 3 根。
   - 从 0 点低点到高点涨幅不少于 8%。
3. 当 EMA10 转为下降时，如果上面条件成立，则确认 1 点。
4. 图表会在高点位置显示“1出”。

所以要区分：

- 1：结构高点的位置。
- 1出：系统确认这段上涨结束、阶段高点成立的时间。

## 4. Gann Box 水平线

确认一组 Gann Box 后，只保留最新一组 5 条线：

- 0
- 0.382
- 0.5
- 0.618
- 1

这些线用于观察支撑、压力、回撤和突破。

## 5. 图表 BUY 点定义

图表中目前有 BUY A、BUY B、BUY C 三类，并且可以单独开关。

### BUY A：早期启动点

偏左侧，核心是低位启动。

触发逻辑包括：

- EMA10 上拐导致 0 出现；或
- RSI 在低位区间上拐；
- 同时要求 RSI 高于默认下限 35；
- 默认要求近期出现过绿柱；
- 价格不能低于当前结构低点。

BUY A 的意义：尝试捕捉早期反弹启动，但波动会更大。

### BUY B：4H 反弹买点

当前图表版本中，BUY B 已经去掉原来的 Gann 回踩反弹逻辑，主要基于 4H RSI 低位反转。

默认逻辑：

- 4H RSI 在低位区间内反转上拐；
- 默认低位区间大致为 35 到 55；
- 当前价格表现不能太弱，例如收阳或高于前一根收盘；
- 如果已有 Gann 0，则价格需要在 0 之上。

BUY B 的意义：比日线更及时地捕捉反弹。

### BUY C：突破确认点

偏右侧，核心是突破确认。

默认逻辑：

- 价格向上突破当前 Gann 1 线；
- MA10 > MA20；
- RSI 高于默认阈值 55；
- 带冷却周期，避免连续重复提示。

BUY C 的意义：确认趋势重新走强，但位置通常比 BUY A / BUY B 更高。

## 6. OneClick Dashboard 新增 BUY/SHORT 逻辑

Dashboard 主要用于批量跟踪，不改变图表本身。

### BUY B：日线分金低位 + 4H 0出

信号名称：BUY_B_D1FJ_low_H4_0birth

逻辑：

1. 日线寻龙诀之分金处于低位背景：
   - 当前分金值 <= 40，或
   - 最近 5 日内分金最低值 <= 40。
2. 日线分金不能继续明显恶化：
   - 当前分金值 >= 前一日分金值，或
   - 最近 5 日内曾经低到 10 以下，属于深度低位。
3. 4H 出现 Gann 0出。

解释：

日线说明位置已经偏低，4H 的 0出说明短周期开始出现反弹结构。这个信号适合寻找反弹型买点。

### SHORT A：日线分金转弱 + 4H 1出

信号名称：SHORT_A_D1FJ_weak_H4_1birth

逻辑：

1. 4H 出现 Gann 1出。
2. 同时满足以下任意一种日线转弱背景：
   - 日线分金开始下降；
   - 日线价格走弱，例如收盘低于前一日或跌破 EMA8；
   - 最近 8 日内日线分金曾经高于 60，说明处于相对高位后转弱。

解释：

4H 的 1出表示短周期反弹高点确认；如果日线背景也开始转弱，就可以作为做空或减仓观察点。

## 7. 红柱、紫柱和分金的意义

寻龙诀之分金里的柱体用于观察情绪和强弱变化：

- 低位黄色/紫色区域：表示情绪偏低，可能处于修复前夜。
- 绿柱：常用于观察低位修复或资金重新活跃。
- 红柱/紫柱：常用于观察高位转弱、动能衰减或风险释放。

在当前系统里，红柱和紫柱不是唯一条件，而是重要加分项。尤其是做空点，如果 4H 出现 1出，同时日线或分金出现红柱/紫柱，高点确认的参考价值更高。

## 8. 使用方式建议

比较稳妥的理解方式：

- BUY A：早期低位启动，激进。
- BUY B：4H 反弹买点，当前最适合做反弹跟踪。
- BUY C：突破确认，偏右侧。
- SHORT A：4H 高点确认 + 日线转弱，适合做空或风险控制。

不要把单个信号当成独立交易指令。更好的用法是结合：

- 大盘环境；
- 个股位置；
- 成交量；
- 支撑压力；
- Gann 0/0.382/0.5/0.618/1 线的位置。

## 9. 当前默认参数

Gann Box 默认参数：

- EMA 长度：10
- EMA 连续上升最少根数：3
- 低点到高点最小涨幅：8%
- Gann 水平线：0、0.382、0.5、0.618、1

OneClick 新增策略默认阈值：

- 日线分金低位：<= 40
- 日线深度低位：最近 5 日最低 <= 10
- 日线高位转弱参考：最近 8 日最高 >= 60
- 4H 触发：Gann 0出 或 Gann 1出

## 10. 风险说明

该指标用于技术分析和交易辅助，不构成投资建议。任何买入、做空或减仓动作都应结合个人风险承受能力、仓位管理和市场环境判断。
